# Server for `NTU Library Study Space`

This is a apollo-express server

## Build Setup

##### mongodb
First you need a mongodb running,

open a separate terminal to run:
```
mongod
```

##### connect to mongo shell
to debug, run

```
mongo library
```

collections are called `student` and `record`

```
# in mongo shell:
db.student.find()
db.record.find()
```
